

Las artes de las peleas varian mucho estan del cuerpo a cuerpo hasta katanas

Son fáciles de reconocer


# Sistema y variantes


Kenjutsu: pelea con katanas dagas o chuchillos

Gōjutsu: pelea con entorno osea usar objetos cosas materiales para defenderse atacar o matar 
Eje: una rama para detener una patada

Hakajutsu: pelea cuerpo a cuerpo puño limpio sin objetos nada
